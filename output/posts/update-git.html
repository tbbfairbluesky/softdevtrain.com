    Update Git

    Software Development Training, LLC 2018-03-14 13:03

    Table of Contents

    • Assumptions
    • Useful Links
    • Steps

    Assumptions

    • You are using Git from Ubuntu/Debian Linux

    Useful Links

    • Stackoverflow question about saving credentials
    • Git Credentials Documentation

    Steps

    1. Add the git-core Personal Package Archive (PPA) repository

      $ sudo add-apt-repository ppa:git-core/ppa

    2. Download package lists for repositories and update them to get newest versions of packages

      $ sudo apt-get update

    3. Install the latest version of git

      $ sudo apt-get install git
