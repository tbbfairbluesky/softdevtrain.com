    Update and Upgrade After Initial Boot-up

    1. Start a terminal window (Ctrl - Alt - T)

    2. Enter the following commands

      $ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install -f

    Remove Amazon Icon from Launcher

    1. Right click on Amazon Icon
    2. Select Unlock from Launcer in context menu

    Install Dropbox

    1. Visit Install Dropbox

    2. Download Ubuntu (.deb) 64-bit

    3. Open the .deb with Ubuntu Software Center

    4. Once Dropbox is installed, run Dropbox and configure it

      Select the Dropbox icon in the upper right and choose Preferences...

      Set the following preferences

    General

    • Check the Start Dropbox on system startup box

    Account

    • Link the computer to your Dropbox account

    Bandwidth

    • No changes from defaults

    Proxies

    • No changes from defaults

    Notifications

    • Uncheck Notify me about * New files synced * Edits to files

    Sync

    • Move Dropbox folder to ${HOME}/Documents

      This should end up placing the files in ${HOME}/Documents/Dropbox

    • Choose Selective Sync... and only sync

      • Apps
      • KeePass2
      • MATLAB
      • Notebooks
      • ubuntu
      • WUSTL

    It may take a couple days before Dropbox folders and files are fully synced.

    Install Zimwiki

    1. In a terminal window, issue the following commands:

    2. Enter the following commands

      $ sudo apt-get update && sudo apt-get upgrade
$ sudo add-apt-repository ppa:jaap.karssenberg/zim
$ sudo apt-get install zim

    3. Open Zim and Open the Following 3 Notebooks

      • WUSTL at ${HOME}/Documents/Dropbox/Notebooks/WUSTL
      • Personal at ${HOME}/Documents/Dropbox/Notebooks/Personal
      • Training at ${HOME}/Documents/Dropbox/Notebooks/Training

    4. Set default notebook to WUSTL

    5. Lock the Zim Desktop Icon to the Launcher and move it up to the top, just below the Dash

    Change Ubuntu Application Menu Location

    1. From the Dash (Ubuntu icon in upper left) enter "settings" and launch the System Settings app. (or use the System Settings icon in the launcher (looks like a gear with a wrench in front of it)
    2. Navigate to Appearance --> Behavior
    3. Under Show the menus for a window choose In the window's title bar

    Configure, Run, and Check Thunderbird

    1. Thunderbird should already be installed as it is a default application for Ubuntu

    2. Before running Thunderbird for the first time

      $ cd
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.thunderbird

    3. Run Thunderbird and check accounts, folders, sending mail, etc.

    4. Lock the Thunderbird Mail icon to the Launcher and move it up to just below the Zim Desktop Wiki icon

    Install and Configure KeePass2

    1. Issue the following commands to install KeePass2

      $ sudo add-apt-repository ppa:jtaylor/keepass
$ sudo apt-get update
$ sudo apt-get install keepass2

    2. Run KeePass2 from the Dash

    3. Open KeePass2 database file - in synced Dropbox

    4. Lock the KeePass2 icon to the Launcher

    Install Chrome

    1. Visit https:://www.google.com/chrome
    2. Select the Download Chrome button
    3. Select 64 bit .deb (For Debian/Ubuntu)
    4. Select Accept and Install

    Set Up Symbolic Links in ${HOME}

    1. Issue the following commands

      $ cd
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/anaconda3
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/awstest
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.bash_aliases
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.bash_logout
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.bashrc
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/bin
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.emacs
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.emacs.d
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/projects
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.ssh
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.thunderbird # This one should be already done.
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/tools
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/workspace
$ cd ~/.config
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.config/zim
$ cd ~/.local/share/applications
$ ln -s ${HOME}/Documents/Dropbox/ubuntu/.local/share/applications/gnome-terminal.desktop

    2. Log out and log back in

    Set Up Terminal Preferences and Profiles

    1. Start a Terminal window

    2. Use the Terminal's menus to navigate to Terminal --> Preferences --> General

    3. Make sure Show menubar by default in new terminals and Enable the menu accelerator key (F10 by default) are checked

    4. Go to the Profiles tab

    5. Rename the Unamed profile to SystemDefault.

    6. Create CHPC1, CHPC2, hcpx-fs01, TimsPreferred profiles as follows

      • CHPC1

        • General: Columns: 132, Rows: 42
        • Colors: Use colors from system theme

      • CHPC2

        • General: Columns: 132, Rows: 42
        • Colors: * uncheck Use colors from system theme * Built-in schemes: Custom * Text color: #000000 * Background color: #FCE9C0 * uncheck Use transparency from system theme

      • hcpx-fs01

        • General: Columns: 132, Rows: 42
        • Colors: * uncheck Use colors from system theme * Built-in schemes: Black on light yellow * uncheck Use transparency from system theme

      • TimsPreferred

        • General: Columns: 132, Rows: 42
        • Colors: * uncheck Use colors from system them * Built-in schemes: Solarized light * uncheck Use transparency from system theme

    7. Set defaut to TimsPreferred

    8. Test open terminals on all platforms

    Change Desktop Background

    1. System Settings --> Apperance
    2. Change from Wallpapers to Colors & Gradients
    3. Select Color Gradient with down indicator, "v"
    4. Select Left color and set to pre-prepared light blue color
    5. Right Color stays black

    Install Emacs

    1. Issue the following command

      $ sudo apt install emacs24-nox

    Install Geany

    1. Issue the following commands

      $ sudo apt install geany
$ sudo add-apt-repository ppa:geany-dev/ppa
$ sudo apt-get update
$ sudo apt-get install geany geany-plugins

    2. Run Geany from the Dash

    3. Lock the Geany icon in the Launcher

    Install Filezilla

    1. Issue the following commands

      $ sudo apt-get update
$ sudo apt-get install filezilla

    2. Add the following sites to the FileZilla Site Manager

      • fastmail ftp
        • Host: ftp.fastmail.com
        • Protocol: FTP
        • Logon Type: Normal
        • User: Get from KeePass2
        • Password: Get from KeePass2
      • hcpcourse machine as admin
        • Host: 128.252.155.182
        • Protocol: SFTP
        • Logon Type: Normal
        • User: Get from KeePass2
        • Password: Get from KeePass2
      • hcpcourse machine as hcpcourse
        • Host: 128.252.155.182
        • Protocol: SFTP
        • Logon Type: Normal
        • User: Get from KeePass2
        • Password: Get from KeePass2

    3. Lock FileZilla to Launcher

    Verify Anaconda Python 3 Environment

    1. Issue the following commands:

      $ source activate python3

    2. Make sure this gets you the Anaconda Python 3 environment with Nikola version v7.8.12 or higher installed

      $ nikola --version

    Install Gnome2 Tools

    1. Issue the following commands

      $ sudo apt install libgnome2-bin

      This will give you access to commands line gnome-open

    Install gitg

    1. Use the Ubuntu Software Center and search for gitg
    2. Install it
    3. Run it and make sure the icon is locked to the launcher

    Install Unity Tweak Tool

    1. Use the Ubuntu Software Center and search for Tweak
    2. Install the Unity Tweak Tool instead of the Tweak Tool
    3. Unlock it from the launcher

    Configure Nautilus

    1. See How to Easily Add Custom Right-Click Options to Ubuntu's File Manager

    2. Install nautilus-actions

      $ sudo apt-get install nautilus-actions

    3. Log out and log back in in order to restart the Nautilus file manager.

    4. Run nautilus-actions from the Dash

    5. Add PrependModDate action - replace ${HOME} with actual path to home directory

      • Action
        • Check Display item in selection context menu
        • Check Display item in location context menu
        • Context Label: PrependModDate
      • Command
        • Path: ${HOME}/bin/PrependModDate
        • Parameters: %B
        • Working Directory: %d
      • Execution
        • Execution Mode: Normal

    6. Add RemoveNumbers action

      • Action
        • Check Display item in selection context menu
        • Check Display item in location context menu
        • Context Label: RemoveNumbers
      • Command
        • Path: ${HOME}/bin/remove_numbers
        • Parameters: empty
        • Working Directory: %d
      • Execution
        • Execution Mode: Normal

    7. Set up Nautilus Preferences

      • In Nautilus: Edit --> Preferences

        • Views * View new folders using: List View * Arrange Items: By Name * Check: Sort folders before files * Check: Show hidden and backup files
        • Behavior * Double click to open items * View executable text files when they are opened * Ask before emptying the Trash or deleting files
        • List Columns * Name, Size, Type, Modified
        • Preview
          • Files
            • Show thumbnails: Never
          • Folders
            • Count number of items: Local Files Only

    Configure Date and Time Options

    1. Find the Time & Date settings in the System Settings App
    2. Select the Clock tab
    3. Check the following options
      • In the clock, show Weekday
      • In the clock, show Date and month
      • 24-hour time

    Set Up for Mounting Remote File Systems

    1. Install sshfs

      $ sudo apt update
$ sudo apt install sshfs

    2. Create remote mount directory

      $ cd ~
$ mkdir -p mnt/chpc2
$ mkdir -p mnt/fs01

    3. Test mount CHPC2 file system

      $ mount_chpc2
$ cd ~/mnt/chpc2
$ ls

    4. Test mount fs01 file system

      $ mount_fs01
$ cd ~/mnt/fs01
$ ls

    Install Search for Files Tool

    1. Install recoll

      $ sudo apt-get install recoll

    2. Install catfish

      $ sudo apt-get install catfish

    3. Run each to make sure recoll is indexing and to lock catfish icon to Launcher

    Update to Latest Version of LibreOffice

    1. Issue the following commands:

      $ sudo add-apt-repository ppa:libreoffice/ppa
$ sudo apt update
$ sudo apt install libreoffice
